# PR-O 법무 자문 의뢰 brief (₩30~50만)

> **대상**: 사장님 → 외부 변호사 (부동산 / 개인정보 전문)
> **목적**: 권리분석 + 결제 freemium 출시 전 법적 컴플라이언스
> **선행 RFC**: 0018 마스터플랜

---

## 1. 자문 범위

### 1.1 권리분석 면책 (필수)
**상황**: WISHES 가 등기부등본 자동 분석 + 위험 알림 (RFC 0018 §2.2).

**자문 필요**:
- 분석 결과가 잘못되어 사용자 손해 발생 시 책임 범위
- 면책 조항 작성 (서비스 약관 §X "참고용 정보, 법적 자문 대체 X")
- "사장님 검토 단계" 가 책임 분산 효과 있는지

### 1.2 등기부등본 발급 대행 (필수)
**상황**: CODEF API 통해 등기부 자동 fetch + 사용자 PDF 제공.

**자문 필요**:
- 위임장 / 사용자 동의서 양식 (전자 동의 OK?)
- 등기부 데이터 보관 기간 (개인정보보호법 §21)
- PDF 만료 정책 (30일 후 자동 삭제 OK?)

### 1.3 결제 약관 + 환불 정책 (필수)
**상황**: Toss Payments ₩3,000 결제 → 등기부 발급 실패 / 분석 오류 시 환불.

**자문 필요**:
- 자동 환불 트리거 조건 (등기부 발급 실패 / 5분 초과 / 분석 오류)
- 부분 환불 (분석 일부만 정확) 처리
- 청약철회 (전자상거래법 §17 — 7일 청약철회 vs 디지털 즉시 사용)

### 1.4 개인정보처리방침 갱신 (필수)
**상황**: 등기부 / 결제 / VAPID 푸시 등 신규 PII 처리.

**자문 필요**:
- 처리방침 §X 등기부 처리 항목 추가
- 정보주체 권리 (열람 / 삭제 / 처리정지) 절차
- 위탁업체 (CODEF / Toss / Resend / Vercel / Supabase) 명시

### 1.5 부동산 광고 (KISO 14항 검증)
**상황**: 매물 등록 + KISO 14항 자동 검증 trigger (이미 구현).

**자문 필요**:
- 자동 검증 누락 시 (사장님 14항 누락 매물) 책임
- 위반건축물 표시 의무 (공인중개사법 §25-2)
- 허위 매물 신고 처리 (KISO 운영지침)

## 3. 의뢰 방법 (사장님 결정)

### 옵션 A — 단발 자문 (₩30만, 추천)
- 변호사 1명 (부동산 전문) 1회 미팅 (2시간)
- 필요 시 후속 이메일 1-2회
- 결과: 약관 / 면책 조항 초안 + 법적 리스크 매트릭스

### 옵션 B — 패키지 자문 (₩50만)
- 변호사 1명 + 개인정보 전문 1명
- 미팅 2회 + 약관 작성 + 검토
- 결과: 즉시 출시 가능한 약관 / 처리방침 / 면책 조항 완성본

### 옵션 C — 자체 작성 + 검토 (₩10만)
- 사장님 + Claude 가 약관 초안 작성
- 변호사 1명 검토 1회
- 결과: 비용 절감, 깊이 부족 (출시 후 수정 가능성 大)

### 옵션 D — 보류
- Phase 2.A (무료 tier 강화) 만 우선 진행
- 결제 / 권리분석 (Phase 2.B/C) 출시 직전 자문

**추천**: 옵션 A (Phase 2.A 진행하면서 병행).
